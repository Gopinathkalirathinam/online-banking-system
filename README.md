Project Overview

A simple console-based online banking application implemented in Java.
Customers can view their balance, deposit money, and withdraw money. Bank employees can log in to the portal and process transactions. This project is ideal as a starter/demo application for learning JDBC, basic authentication, and simple transaction handling with a MySQL backend.

Tech Stack

Language: Java (SE)
Database: MySQL
DB Access: JDBC
IDE: Eclipse (or any Java IDE / command line)
Build: Plain javac / java or your preferred build tool

Features:
Customer operations:
View account balance
Deposit money
Withdraw money (with balance checks)
Employee operations:
Employee login (simple username/password)
View and process customer transactions
Persistent storage using MySQL
Proper error handling for invalid inputs and DB errors
Simple console-based menu-driven UI
